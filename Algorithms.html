Web application Algorithms
Map with 50000 points and Cluster Algorithm

each point can be created, read, updated, deleted (crud) from MSSQL DB. every tooltip loaded in ajax request from DB.

Graph paging algorithm

Graph paging 100000 points algorithm - in next/previous click user get next/previous 100000 results on the graph ( from PL/SQL DB ) - written from scratch.

Offline Java map server

in each map zoom event the GeoServer generate new WMS file and the client generates the current map tiles ( 256x256 pixels images that assemble full map view).

Squere lasso

the highcharts API doesn't have lasso - so lasso is written from scratch on existing API.

Square lasso is written on default zoom controller .

Polygon lasso Algorithm

the user draw a polygon, the client send the polygon points to the server , the server calculates and returns all the graph points that are inside that polygon (written from scratch).
